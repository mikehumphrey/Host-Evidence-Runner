## 3 EASY STEPS

### 1. PLUG IN USB
- Insert USB drive into server

### 2. RUN TOOL
- Open File Explorer
- Find USB drive folder
- **Double-click `RUN_ME.bat`**
- Click "Yes" if Windows asks

### 3. WAIT & RETURN
- Watch progress window
- Takes 15-30 minutes
- When done, take USB back
- Return folder to analyst

---

## TROUBLESHOOTING

**Error: "Administrator Required"**
→ Right-click RUN_ME.bat → Run as administrator

**Error: "PowerShell Disabled"**
→ Contact IT - PowerShell needed

**Window Closes Quickly**
→ Run Command Prompt as admin → Go to USB → Type: RUN_ME.bat

**Takes Too Long (>1 hour)**
→ Normal on large servers - let it finish

**Antivirus Warning**
→ Safe to allow - only reads files
